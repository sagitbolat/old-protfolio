sPiet: a Piet interpreter

"Piet" is an esoteric programming language created by David Morgan-Mar that aims to convert the form of code into abstract art. This project is an interpreter for a custom dialect of piet, with minor differences from the original specifications. The goal of the dialect is to improve readability of the language, and introduce custom palettes.

Conway's Game of Life

Conway's game of life is a cellular automaton. It requires no input apart from its inital board state. The player simply sets up the game and watches it play itself. The game is turing complete, meaning it is possible to create programs inside the game itself. The following project is an implementation of the game, with several variations on the original rules.
